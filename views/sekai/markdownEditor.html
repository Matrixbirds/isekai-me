在文字前加入若干#可以使之成为标题：

# 一级菜单
## 二级菜单
### 三级菜单
#### 四级菜单

也可以在标题下加上一串---或者===来标识一级和二级菜单：

一级菜单
-------
二级菜单
=======

				
Markdown中可以使用以下文字效果：

*斜体*
**加粗**
***加粗和斜体***
~~删除线~~

				
Markdown中的无序列表是在*、+或者-后加个空格产生：

* 无序列表1
* 无序列表2
* 无序列表3

Markdown中的有序列表是在序号后加个点和空格产生：

1. 有序列表1
2. 有序列表2
3. 有序列表3

列表可以拥有层级关系，需要在前面加上空格：

* 无序列表1
 * 无序列表1-1
 * 无序列表1-2

				
Markdown中的引用可以在>后面加空格产生：

> 引用内容

引用可以控制层级，通过添加多个>产生：

> 引用内容
>> 引用内容中的引用内容

				
Markdown中的链接如下格式：

[link text](http://example.com/ "optional title")

这里link text表示链接的alt值，http://example.com/表示链接地址，optional title表示链接的title值。
也可以使用编号管理：

[aaa][1]
[bbb][2]
[1]: http://example.com/aaa
[2]: http://example.com/bbb

图片如下格式：

![link text](http://example.com/xxx.jpg "optional title")

这里link text表示图片的alt值，http://example.com/表示图片地址，optional title表示图片的title值。

Markdown中的行内代码可以用`包裹起来：

这是一般文字，`这是代码`，这又是一般文字。

你也可以使用```将代码块包裹起来使用，并标明语言类型用于高亮：

```javascript
    // TODO
```

				
Markdown中的表格使用|作为单元格分割，并使用---来区分表头：

| Name | Age | Class |
| ---- | --- | ----- |
| Bill | 12  |   5   |
| Lily | 13  |   5   |
| Lucy | 12  |   5   |

当然，你不必对齐|，只需要在每个单元格前后加上空格即可：

| Name | Age | Class |
| ---- | --- | ----- |
| Bill | 12 | 5 |
| Lily | 9 | 5 |
| Lucy | 12 | 11 |

你可以在表头控制一列文字的定位——居中，靠左以及靠右：

| Name | Age | Class |
|:----:|:--- | -----:|
| Bill | 12 | 5 |
| Lily | 9 | 5 |
| Lucy | 12 | 11 |
